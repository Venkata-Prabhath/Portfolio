## Installation

Clone the repository

```bash
git clone https://github.com/Venkata-Prabhath/Portfolio.git
```

Go to project folder

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
